Команды для админов:
/daily_check - сделать срочную проверку счетов (высылается напоминание для должников / проходят оплаты в день сбора)
/list_mappings - вывести таблицу сопоставлений никнеймов и id и юзернеймов
/set_fee <new_fee> - выставить новое значение суммы сбора
/set_payment_day <new_day> - выставить новое значение дня месяца сбора
/confirm_payment <tg_id|nickname> <amount> - подтвердить перевод, добавить деньги на счёт человека в боте
/add_key <tg_id|nickname> <tag|None=_> <key_text> - добавить ключ человеку или добавить в очередь на добавление, если он еще не зарегистрировался (в таком случае ключ будет мгновенно добавлен при регистрации чела)
/generate_invite <None|nickname> - сгенерировать инвайт код для нового пользователя. Желательно указать никнейм для нового человека для удобной работы
/broadcast <text> - сделать объявление всем пользователям бота
